## TIME VARIABLES

{hour:1} - Hour in 24-hour format (00-23)
{hour:2} - Hour in 12-hour format (01-12)
{min:1} - Minutes (00-59)
{min:2} - Minutes with AM/PM (e.g., "30 PM")
{time:24hr} - Time in 24-hour format (20:23)
{time:12hr} - Time in 12-hour format with AM/PM (8:23 PM)

## THAI DATE VARIABLES

{th=date} - Day of month (1-31)
{th=week:1} - Abbreviated weekday (e.g., "จ.", "อ.")
{th=week:2} - Full weekday (e.g., "จันทร์", "อังคาร")
{th=month:1} - Month number (1-12)
{th=month:2} - Abbreviated month (e.g., "ม.ค.", "ก.พ.")
{th=month:3} - Full month (e.g., "มกราคม", "กุมภาพันธ์")
{th=year:1} - Thai year, 2 digits (e.g., "68")
{th=year:2} - Thai year, 4 digits (e.g., "2568")

## ENGLISH DATE VARIABLES

{en=date} - Day with suffix (e.g., "1st", "2nd", "3rd")
{en=week:1} - Abbreviated weekday (Mon, Tue, Wed)
{en=week:2} - Full weekday (Monday, Tuesday)
{en=month:1} - Month number (1-12)
{en=month:2} - Abbreviated month (Jan, Feb, Mar)
{en=month:3} - Full month (January, February)
{en=year:1} - Year, 2 digits (e.g., "25")
{en=year:2} - Year, 4 digits (e.g., "2025")

{avghumidity} - Average humidity as percentage
{wind:kph} - Wind speed (km/h) 3.6

{precip:mm} - Rainfall / Units millimeter mm (e.g., "3.65")

{windchill:c} - Wind chill temperature / Celsius °C (e.g., "20.7")
{windchill:f} - Wind chill temperature / Fahrenheit °F (e.g., "69.3")
{heatindex:c} - Heat index / Celsius °C (e.g., "20.7")
{heatindex:f} - Heat index / Fahrenheit °F (e.g., "69.3")

{vis:km} - Visibility / Kilometer (e.g., "4.5")

{humidity} - Humidity / Percentage % (e.g., "52")

{uv} - UV index / Index (e.g., "1")
